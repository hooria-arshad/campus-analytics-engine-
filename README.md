# campus-analytics-engine-

**Author:** Hooria Arshad

## Overview
Campus Analytics Engine is a menu-driven C++ application developed as a Programming Fundamentals project. It manages student records, course enrollments, attendance, grades, fee tracking, and report generation using text files for data storage.

## Features
- Student Management
- Course Enrollment
- Attendance Tracking
- Grade Management
- Fee Tracking
- Report Generation
- File Handling

## Technologies Used
- C++
- File Handling
- Functions
- Structures
- Modular Programming

## Compile

```bash
g++ -o campus_engine main.cpp filehandler.cpp student_ops.cpp course_ops.cpp attendance.cpp grades.cpp fee_tracker.cpp reports.cpp
```

## Run

```bash
.\campus_engine.exe
```

## Project Purpose
This project was created as part of the Programming Fundamentals course to demonstrate file handling, modular programming, data processing, and report generation in C++.
